
# 线性结构

上一节我们讲了数组，数组的插入和删除可以发生在任何位置  
我们将下标为0的这一头称为头部front，另一头称为尾部back。  
栈stack是插入和删除都只能发生在尾部（栈顶），队列queue是只能在尾部插入和头部删除。进一步的，如果同时允许在头部和尾部进行插入和删除，称为双端队列(double-ended queue, deque)
